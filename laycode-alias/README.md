# laycode

> `laycode-cli` 的别名包 — 运行 LayCode 桥接服务。

`laycode` 与 [`laycode-cli`](https://www.npmjs.com/package/laycode-cli) 完全等价，只是提供一个更短的命令名。所有参数、命令、行为都一致。

## 用法

```bash
npx laycode
```

等同于：

```bash
npx laycode-cli
```

完整命令与选项见 [`laycode-cli` 文档](https://www.npmjs.com/package/laycode-cli)。
