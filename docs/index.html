opencode 的移动端客户端

躺着码，一样 Vibe

在手机上继续你的 opencode 编码会话。通过运行在电脑上的桥接服务， 手机 App 安全连接本地 opencode，随时随地继续对话、查看代码、管理任务。

下载 Android App 在 GitHub 查看
电脑端一行命令启动桥接服务
$ npx laycode-cli

为移动端打造

把桌面的编码能力，装进你的口袋。

会话同步

无缝接续你在电脑上的 opencode 对话，换个屏幕继续聊。

内置终端

基于 xterm.js 的完整终端模拟器，在手机上直接跑命令。

扫码即连

扫描桥接服务的二维码，几秒钟完成配对，无需手动输地址。

自动发现

mDNS 自动发现同一局域网内的桥接服务，连接省心。

任务管理

按工作区整理的个人待办清单，勾选完成、标记紧急，一目了然。

安全连接

Token 认证，所有请求经桥接服务代理，数据不出局域网。

界面一览

深色、专注、为编码而生。

快速开始

两步，手机连上你的电脑。

1

电脑端启动桥接服务

需要 Node.js ≥ 18。运行下面的命令，终端会显示配对二维码。

$ npx laycode-cli
2

手机端扫码连接

下载并打开 LayCode App，扫描二维码即可连接（确保手机和电脑在同一 Wi-Fi）。
